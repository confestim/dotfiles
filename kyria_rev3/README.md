Still learning how to use this keyboard, but it has been quite nice so far.

*Files:*

- vial/ is the keyboard firmware, which is to be placed in the `qmk_firmware/keyboards/` directory.
-  layout.vl is the layout file for Vial, which is to be loaded in vial


*My changes:*
I've mostly just been changing the vial config, but I did add some cool OLED stuff to the firmware (LoL and csgo icons for respective layers, a pi otherwise). I realize the pro micro is way too small, so I advice against using it...
